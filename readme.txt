== Description ==

Custom Checkout works only when the users visits from any of the discounted Links 

Airsuggest WooCommerce features:

* Customized Thank you page message
* Social Media - Give exclusive discounts to Influencers - 
* API Key, if you will not provide api key than it will return error.
* The data will be automatically transferred to Airsuggest - https://www.airsuggest.com/m/api_checkout.php

Ask any query e-mail: admin[at]airsuggest.com

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/woo-airsuggest' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Airsuggest WooCommerce payment gateway -> Settings screen to configure the plugin
